Sobre mim

Olá, sou Diego Castro, nesse momento me encontro em transição de carreira da área de infraestrutura para a área de Desenvolvimento Web.

Tenho experiência em desenvolvimento web com as principais tecnologias HTML, CSS3, JavaScript, Web Design Responsivo e frameworks React.Js e Node.Js. Crio interfaces envolventes e funcionais, busco constantemente desafios inovadores para aprimorar minhas habilidades.

HTML | CSS | GIT | JAVASCRIPT | NODE.JS | REACT.JS | TYPESCRIPT | REACT NATIVE | PYTHON
